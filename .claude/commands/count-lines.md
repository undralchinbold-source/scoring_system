Count lines in $ARGUMENTS and report a summary.

## Шалгах зүйлс

1. Файлын нийт мөрийн тоо
2. Хоосон мөрийг хассан тоо
3. `def ` тоо (функцийн тоо)
4. `class ` тоо
5. Хамгийн урт функц хэдэн мөртэй вэ

## Хариултын формат

```
Файл: <зам>
Нийт: N мөр
Хоосон бус: N мөр
Функц: N
Class: N
Хамгийн урт функц: <нэр> (N мөр)

Дүгнэлт: <файл хэт урт уу? функц хэт урт уу?>
```

Хязгаар: файл 200 мөрөөс бага, функц 20 мөрөөс бага байх ёстой.
Хязгаар хэтэрсэн бол ❌ тэмдэгтэй харуулна.
